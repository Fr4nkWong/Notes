## 前言
Git就是代码的分布式管理系统。

## 常用命令&使用场景
0x01: 上传/更新代码
``` bash
git clone ***.git (目录下已存在.git目录可跳过此行命令)
git add .
git commit -m "commit content"
git remote add origin <git ssh code>
git push -u origin master
```
